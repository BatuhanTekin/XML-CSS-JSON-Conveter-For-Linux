# XML-CSS-JSON-Conveter-For-Linux
XML/CSS/JSON Conveter For Linux
 GOAL
 
   Convert files between CSV,XML,JSON formats.
   
  
  How to run:
  
  
    converter.c <input file> <output file> <type>
    
    sample : converter.c text.xml test.json 3
    
    
    Types :
    
    1-CSV TO XML
    2-XML TO CSV
    3-XML TO JSON
    5-CSV TO JSON
    6-XML TO XSD
